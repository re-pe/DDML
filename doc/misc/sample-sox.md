# SOX

Kodo pavyzdys iš [SOX][wiki:sox]

XML kodas

```xml
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
  <title>Sample page</title>
</head>
<body>
  <p>A very brief page</p>
</body>
</html>
```

SOX kodas

```sox
html>
    xmlns=http://www.w3.org/1999/xhtml
    head>
        title> Sample page
    body>
        p> A very brief page
```
